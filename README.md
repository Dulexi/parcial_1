# Proyecto Parcial 1 - DevOps

Proyecto práctico desarrollado para la evaluación parcial de la unidad didáctica DevOps: Fundamentos.

## Estructura


- src: código Python.
- datos: archivos de entrada.
- reportes: resultados generados.
- evidencias: capturas del laboratorio.



## Ejecución del proyecto

Para ejecutar el procesamiento del inventario:

python3 src/procesar_inventario.py


El sistema genera los siguientes reportes:

- reportes/reporte_inventario.csv
- reportes/resumen_inventario.json

## Funcionalidades implementadas

- Lectura de datos en formato JSON.
- Validación de campos obligatorios.
- Validación de direcciones IP.
- Generación de reporte CSV.
- Generación de resumen JSON.
- Manejo de errores mediante excepciones.
